API

Create

HTTP POST /api/user/:userfields

Read

All: HTTP GET /api/users

By ID: HTTP GET /api/user/:_id

Update

HTTP POST /api/user/:_id

passes req.query as json object into mongoDB update ($set: req.query)

Delete

HTTP DELETE /api/user/:_id


User login paths

login page GET /api/login

renders login page. Will add error message on failed or expired login

login submission POST /api/login

Submits user login, creates cookie if login is valid. Redirects to user home


/home

HTTP GET /home

Renders user page from cookie, if no cookie present will redirect to login
